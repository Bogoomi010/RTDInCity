## 🎯 게임 방향성 (톤 바이블)

**타깃**: 10대~30대 남녀 · 캐주얼

**핵심 정서**: "도심 속에서 누구나 한 번쯤 봤거나 겪어봤을 법한" 캐릭터로 **공감**을 만든다.
전단지 알바, 붕어빵 사장, 길고양이, 비둘기, 취준생, 야근하는 대리 — 특별한 영웅이 아니라
**우리 동네에 있는 그 사람**이 주인공이다.

**스토리 공식** — 평범한 도시인의 사연이 **살짝 막장·병맛으로 각성**하는 순간을 담는다:

> 예) 전단지 알바 → "아이폰이 갖고 싶어 시작한 전단지 알바, 어느 날 **던져서 정확히 맞추는 재능**에 눈을 뜬다."

- 톤: **코믹하고 과장되게**, 하지만 밑바탕엔 진짜 공감(월세·야근·취준·인간관계)이 깔린다
- 등급이 오를수록 = **직업적 출세 + 사연의 스케일업** (신입 경관 → 경찰청장, 은행 인턴 → 재벌 총수)
- 계열은 "한 사람의 커리어 성장 서사" 혹은 "같은 세계의 급 다른 존재들"로 읽히게 한다
- 초월 = 두 인생이 만나 도시의 신화가 되는 순간 (시장 + 경찰청장 = 시티 로드)

**금지**: 진지·비장·잔혹, 정치·종교·특정 인물 풍자, 현실 브랜드 실명.
"웃기지만 미워할 수 없는" 선을 지킨다.

## ⚔ 전투 축 기획 — 공격 타입 & 유닛 타입

> 상태: 📝 **기획 (미구현)**. 등급별 문서의 표가 유닛별 판정의 원본이다.
> 구현 시 `UnitDef`에 `atkType: "melee" | "ranged"`, `air?: boolean` 필드를 추가하고
> `balance-sim.ts`로 재검증 후 GDD 1.7을 갱신한다.

### 공격 타입 — 원거리 / 근거리

| | 원거리 🏹 | 근거리 ⚔ |
| --- | --- | --- |
| 컨셉 | 던지고 쏘는 직업 | 몸으로 부딪히는 직업 |
| 사거리 | 표준 (동급 기준) | 동급 원거리 대비 **-30%** |
| DPS 예산 | 표준 | 동급 원거리 대비 **+15%** |
| 배치 | 트랙 안쪽 어디든 무난 | **도로 인접 배치 필수** — 배치 퍼즐 담당 |

- 근거리는 "짧은 사거리 + 높은 화력"의 하이리스크·하이리턴. 코너 안쪽(도로 2면 인접)이 최적 배치
- 공격 모션도 타입을 따른다: 원거리 = 투척/사격, 근거리 = 스윙/찌르기 (아트 프롬프트에 반영)

### 유닛 타입 — 지상 / 공중

| | 지상 🚶 | 공중 🕊 |
| --- | --- | --- |
| 이동 | **적군 경로(도로)를 지나갈 수 없다** — 현행 트랙 안쪽만 | **도로를 자유롭게 넘어다닌다** — 도로 위 호버링·보도(트랙 바깥) 배치 가능 |
| 연출 | 현행 | 부유 모션(상하 ±4px) + 발밑 분리 그림자 |
| 밸런스 | 표준 | 배치 자유의 대가 — DPS 예산 **-10%** (드론 계열은 기존 마법 페널티에 포함된 것으로 간주, 중복 적용 안 함) |

- 공중 판정 기준: **유닛의 전투 주체가 나는 존재**인가 (드론 계열 전체 = 조종사가 아니라 드론이 본체, 닭둘기 = 새)
- 조합 결과 유닛은 **지배 재료의 공격 타입·유닛 타입을 상속** (`combos.ts`의 SHAPES에 필드 추가)
- 향후 확장 훅: 공중 몹(예: 비둘기 계열 상위) ↔ 대공 전용 유닛 — 지금은 범위 밖

## 🎨 아트 방향 (레퍼런스: 마법상점 Tycoon 2 계열)

- **스타일**: 귀여운 2D 캐릭터, **세미치비(약 1:3 두신)**, 애니메이션풍
- **렌더**: 셀 셰이딩(cel shading), 깔끔하고 굵은 라인아트, 부드러운 그림자
- **색감**: 따뜻하고 채도 높은 팔레트, 은은한 림라이트(rim light)
- **표정/포즈**: 표정이 살아있고 성격이 드러나는 **동적 아이들 포즈**, 소품으로 직업 즉시 식별
- **금지**: 픽셀아트, 3D 렌더, 실사, 과도한 디테일 (게임 내 표시 크기 48×48px 기준 실루엣이 읽혀야 함)
- **배경**: 흰색/투명 (에셋 추출용). 전신 1컷.
- **활용 계획**: 도감 미해금 캐릭터는 이 에셋에 `tint 0x111111`을 씌운 **검은 실루엣**으로 표시 (현재 "???" 자리)

```
2D cute game character, semi-chibi anime style (1:3 head-body ratio),
cel shading, clean bold lineart, warm vibrant color palette, soft rim light,
expressive face, full-body single character, dynamic idle pose,
Korean mobile game / webtoon aesthetic, clear readable silhouette,
white background, high quality, NOT pixel art, NOT 3d render, NOT realistic
```

## 🖼 인게임 스프라이트 프롬프트 (픽셀아트 · 현행 파이프라인)

위의 "아트 방향"(세미치비 셀셰이딩)은 **도감·일러스트용**이고, 실제 인게임 스프라이트는
`fl1` 전단지 알바생 프롬프트를 기준으로 한 **32비트 픽셀아트 2단계 파이프라인**으로 생성한다.
각 등급 문서 하단의 "🖼 인게임 스프라이트 생성 프롬프트" 섹션에 유닛별 전체 프롬프트가 있다.

1. **① 캐릭터 시트**: 큰 버스트 + 정면 전신 + 후면 전신 + 팔레트 칩 (단색 회색 배경)
2. **② 공격 모션**: ①의 결과물을 첨부해 오른쪽 방향 4프레임 공격 시트 생성

**⚠ 크기 통일 규격 (전 유닛 공통 · 절대 규칙)** — 각 문서의 [공통 스타일 블록]에 포함되어 있으며, 절대 빼지 않는다:

- 전신 컷의 캐릭터 높이(발끝~정수리) = 컷/프레임 높이의 **70% 고정**, 발끝 기준선 동일
- 인간형 두신 비율 **약 1:3.5** 통일 — 어떤 시트를 나란히 놓아도 키·머리 크기가 같아 보여야 함
- 동물·기계·부유형도 같은 바운딩 박스 높이를 채운다 — 크기감 차이는 스케일이 아니라 부피·실루엣·이펙트로 표현
- 도트 해상도(픽셀 밀도)·외곽선 두께 전 유닛 동일
- 공중 유닛의 모션 시트는 발 위치 대신 **부유 높이 고정 + 발밑 분리 그림자**
