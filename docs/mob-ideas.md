# 적군(몹) 기획 — 하찮은 것들의 침공

> 아이디어 문서. 채택分은 GDD 파트 2(웨이브 & 보스)와 `src/data/waves.ts`로 옮긴 뒤
> `balance-sim.ts`로 재검증한다.

## 1. 컨셉 원칙

1. **하찮게 시작한다** — 도시를 위협하는 것은 괴물이 아니라 돌멩이, 비둘기, 쓰레기
2. **하찮은 채로 강해진다** — 종족은 그대로, 수식어만 진지해진다 (매우 딱딱한 돌멩이)
3. **끝은 코믹한 신격화** — 슈퍼 돌멩이, 악마 비둘기, 쓰레기의 악마

수비대가 경찰·배달·시장 같은 "진지한 직업"인 것과 대비되어, 도시 전체가
하찮은 것들에게 진심으로 대응하는 코미디가 완성된다.

## 3. 계열 = 카운터 체크 (기존 빌드 축과 연결)

| 계열 | 체크하는 것 | 카운터 빌드 |
| --- | --- | --- |
| 🪨 돌 | 물리 일변도 방지 | 마법(드론) 또는 방깎(철거) |
| 🕊 비둘기 | 순수 DPS 부족 방지 | 감속(교통) + 연타(배달) |
| 🗑 쓰레기 | 단일 딜 일변도 방지 | 광역(스플래시) — 분열 대응 |

라운드 배치는 **3계열 로테이션** (r % 3): 매 라운드 계열이 바뀌어 한 가지 빌드로 방치할 수 없다.
보스 라운드(10/20/30/40)는 기존 보스 4종 유지 — 단, 40R '시티 브레이커'를
"쓰레기의 신"으로 리스킨하는 옵션은 검토 여지 (하찮음 서사의 완성).

## 4. 구현 노트

- `waves.ts`의 `mobStats(round)`에 계열 배율만 곱하면 됨 — 공식 자체는 유지
- 계열 배율 평균이 1.0 근처가 되도록 설계됨 (돌 방어↑ ↔ 비둘기 방어↓) → 전체 난이도 중립
- 분열: `Mob` 사망 시 하위 몹 2기 스폰 (GameScene.damage에서 처리, 필드 캡 카운트 포함)
- 황금 비둘기: 라운드당 ~5% 확률 1기. 트랙 1바퀴 후 자연 소멸(데스 카운트 미차감) — 놓쳐도 벌 없음, 잡으면 보너스 (도파민 전용 몹)
- 비주얼: 텍스처 재활용 + 계열 색(돌 회색/비둘기 하늘색/쓰레기 녹색/황금 노랑) + 단계별 크기 확대. 신규 에셋 불필요
- 시뮬 봇은 armor·hp만 보므로 계열 배율을 `mobStats`에 넣으면 자동 반영됨. 분열·황금 비둘기는 근사(체력 합산/골드 기대값)로 처리
