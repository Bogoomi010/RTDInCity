# 스토리 보스존 기획 — 어느 도시인의 일대기

> 아이디어 문서. 레퍼런스: 원랜디의 "유닛 파견 → 보스 격파 → 보상" 존 구조를
> 도시인의 인생 스토리로 재해석. 채택分은 GDD와 코드로 옮긴 뒤 시뮬 재검증.
> ✅ 10장 일대기·파견 루프·보상 채택·구현 완료 — 확정 스펙은 GDD 파트 5 참고.

## 1. 컨셉

메인 트랙(도시 방어)과 별개로, **대도시에서 살아가는 한 사람의 일대기**를 챕터별
스토리 보스존으로 만든다. 보스는 괴물이 아니라 **인생의 시련**이다 — 몹이 하찮은
사물이듯, 진짜 무서운 것은 이삿짐 상자와 월세 고지서라는 코미디.

배경·정서는 **한국 대도시** (2026-07-03 확정). 수능, 축의금, 치킨집 창업 같은
한국적 소재를 그대로 쓴다 — 정서 곡선(상경→구직→주거→일→가족→은퇴)은 만국
공통이라 외국 플레이어에게도 읽히고, 한국 고유 명사는 이국적 풍미가 된다.
글로벌 출시 시점에 로컬라이징 치환 테이블로 대응 (지금 중립화하지 않음).

## 2. 핵심 루프

1. 필드의 **원하는 유닛 1기를 선택해 스토리 존에 파견** (파견 중엔 수비에서 빠짐 — 기회비용)
2. 파견 유닛의 DPS만큼 스토리 보스 HP가 실시간으로 깎임 (낮/밤 버프 적용)
3. **보스 HP는 유지됨** — 약한 유닛으로 여러 판에 걸쳐 깎아도 됨 (판 간 저장)
4. 격파 시 해당 챕터 보상 획득 + 다음 챕터 해금 → **매 스토리마다 지속적으로 보상**
5. 유닛은 언제든 회수 가능 (빈 칸으로 복귀)

## 3. 챕터 구성 — 일대기 10장

**보상 원칙 — 모든 보스·스토리 보상에는 🎟 소환권 포함** (구현됨: 등급별 소환권 = 그 등급의 원하는 유닛을 지정 소환). 챕터가 오를수록 소환권의 **등급과 수량이 함께 상승**.

| 장 | 제목 | 스토리 보스 | 보상 (🎟 = 소환권) |
| --- | --- | --- | --- |
| 1 | 수능 전날 | 수능 한파 | 골드 +100 + 🎟안흔함 ×1 |
| 2 | 백수 상경기 | 서류 광탈의 벽 | 뽑기 비용 -2G (영구) + 🎟안흔함 ×2 |
| 3 | 첫 출근 | 월요일 아침 2호선 | 🎟특별함 ×1 |
| 4 | 자취 원년 | 월세 고지서 | 골드 +250 + 🎟특별함 ×1 |
| 5 | 대리 진급 | 무한 야근의 화신 | 지원 카드 1장 + 🎟특별함 ×2 |
| 6 | 결혼 대작전 | 축의금 정산서 | 🎟희귀함 ×1 |
| 7 | 내 집 마련 | 대출 이자 드래곤 | 데스 +2 + 🎟희귀함 ×2 |
| 8 | 육아 전쟁 | 새벽 3시의 울음소리 | 히든 유닛 1기 확정 + 🎟희귀함 ×2 |
| 9 | 부장의 무게 | 사내 정치 구름 | 🎟전설 ×1 |
| 10 | 은퇴, 그리고 | 치킨집 창업의 유혹 | 🎟전설 ×2 + 도감 특별 등록 |

- 보스 HP는 챕터마다 지수 증가 (1장: 안흔함 1기가 ~30초, 10장: 초월급 파견 필요)
- 10장 격파 후: **회차 반복** — HP ×3씩 증가, 보상은 골드/카드 위주로 순환 (엔드 콘텐츠)

## 4. 시련 보스의 개성 (선택 구현)

| 보스 | 기믹 |
| --- | --- |
| 수능 한파 | 없음 (튜토리얼) |
| 서류 광탈의 벽 | 물리 저항 50% — 마법 유닛 추천 |
| 월요일 아침 2호선 | 주기적으로 HP 5% 회복 (지각 리스크) — 고DPS 요구 |
| 대출 이자 드래곤 | 시간이 지날수록 HP 재생 증가 (복리!) — 한방 유닛 추천 |
| 새벽 3시의 울음소리 | 🌙 밤에만 피해를 받음 — 밤 태그 유닛 필수 |

## 5. UI

- 메인 화면 좌하단(트랙 바깥 여백)에 **스토리 존 오두막** 아이콘 + 진행 중 보스 HP 바
- 유닛 선택 시 정보 패널에 "📖 스토리 존 파견" 버튼 (파견 중엔 "회수")
- 클릭 시 작은 패널: 현재 챕터 제목, 보스 이름·HP, 파견 유닛, 보상 미리보기
- 챕터 격파 시 전체 화면 토스트: "제3장 완료 — 월요일 아침 2호선을 이겨냈다!"

## 6. 구현 노트

- 상태: `{ chapter, bossHp, sentUnitId }` — 세이브에 저장 (판 간 유지되는 유일한 성장 요소)
- 파견 = 유닛을 `units` 배열에서 스토리 존 슬롯으로 이동 (그리드 셀 해제), DPS 계산은 `def.atk / cooldown × 페이즈 버프`를 초당 적용 — 전투 렌더링 불필요, 숫자만 깎으면 됨
- 회수/격파 복귀 = 빈 칸에 재배치 (빈 칸 없으면 대기 후 자동 복귀)
- 몹 캡·라운드 진행과 완전 독립 — WaveSystem 변경 없음
- 밸런스: 파견 = 수비 화력 손실이므로 보상은 "이득이지만 타이밍 판단이 필요한" 수준으로.
  시뮬 봇은 1차 구현에선 스토리 존을 무시 (보상 = 플레이어 추가 이득 → 봇 승률은 하한선)
