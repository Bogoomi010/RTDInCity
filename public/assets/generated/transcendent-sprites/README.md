# 초월 등급 생성 원본

`docs/characters/6-transcendent.md`의 외형·공격 묘사를 기준으로 imagegen 내장 도구에서 생성한 알파 원본이다.

## 공통 프롬프트 규격

- 용도: `stylized-concept`, 프로덕션용 32비트 시대 픽셀아트 게임 스프라이트 원본
- 스타일: 낮은 채도, 풍부한 명암, 어두운 고정 외곽선, 최소 디더링, 동일 픽셀 밀도
- 비율: 인간형 1:3.5 두신, 기계·부유형도 같은 70% 높이 바운딩 박스
- 색상: 초월 청록 `#26c6da` 에너지와 유닛별 문서 색상
- 배경: 균일한 `#ff00ff` 마젠타 크로마키, 피사체에는 마젠타 사용 금지
- 금지: 글자·숫자·로고·워터마크·격자선·바닥·그림자·반사
- 마스터: 좌측 38% 고품질 버스트, 우측 정면·후면·측면 전신
- 공격: 2행×4열, 준비·축적·발동·후속·복귀가 드러나는 8프레임
- 이동: 정면·후면·측면 각 8프레임, 동일 크기와 기준선

## 런타임 변환

- 크로마 제거: imagegen 스킬의 `remove_chroma_key.py`, 경계 자동 샘플링·소프트 매트·디스필. 마젠타 프린지가 남은 원본은 명시적 `#ff00ff` 키와 1px 가장자리 수축으로 재처리했다.
- 모션: `256x256` 셀, 캐릭터 높이 `180px`, 기준선 `y=244`
- UI: 버스트 `512px`, 패널 초상화 `128px`, 작은 위젯 아바타 `48px`
- `t2`, `t5` 이동 원본은 방향별 7포즈를 검출해 첫 포즈를 복귀 프레임으로 추가했다.
- `t2` 버스트는 인접 전신 포즈가 섞이지 않도록 `--bust-width-ratio 0.30`을 사용했다.

최종 런타임 파일은 `public/sprites/transcendent/`에 있다.
