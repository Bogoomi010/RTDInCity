# 안흔함 등급 생성 원본

`docs/characters/2-uncommon.md`의 외형·공격 묘사를 기준으로 imagegen 내장 도구에서 생성한 원본이다.

## 공통 생성 규격

- 32비트 시대 픽셀아트, 낮은 채도와 어두운 외곽선, 초록 `#4caf50` 포인트 1~2곳
- 인간형 1:3.5 두신, 조종사+드론도 같은 70% 높이 바운딩 박스
- 마스터: 고품질 버스트 + 정면·후면·측면 전신
- 공격: 준비·조준·발동·후속·복귀가 이어지는 2행×4열 8프레임
- 이동: 정면·후면·측면 8프레임, 동일 크기와 기준선
- 배경: 균일한 `#ff00ff` 크로마키, 글자·격자선·바닥·그림자 없음

## 유닛별 핵심

| ID | 외형 | 공격 | 이동 |
| --- | --- | --- | --- |
| `dv2` | 유선형 헬멧·고글·3대 폰·번개 배달통 | 양손 초고속 봉투 투척 | 전경 자세의 빠른 보행 |
| `pc2` | 몸에 맞는 제복·초록 견장·테이저 | 흔들림 없는 양손 정조준 | 여유 있는 순찰 보행 |
| `sn2` | 부시햇·장비 조끼·스코프 공기총 | 호흡 조준과 공기 펄스 | 공기총을 통제한 보행 |
| `dr2` | FPV 고글·조종기·중형 카메라 드론 | 드론 락온·펠릿 2연발 | 조종사와 드론 동기 이동 |
| `dm2` | 안전모·초록 조끼·수동 슬레지해머 | 급소 표시 후 수평 풀스윙 | 해머를 어깨에 멘 중량 보행 |
| `tf2` | 교통모·반사 조끼·흰 장갑·야광봉 2개 | 수신호 초록 파동 | 절도 있는 교통 경찰 보행 |

`dm2` 1차 마스터가 전동 해머로 해석돼 수동 장병기 슬레지해머로 재생성했다. `dv2`, `pc2`, `dm2`의 이동 원본은 7개 고유 자세에 첫 자세를 복귀 프레임으로 연결했다.

## 런타임 변환

- 크로마 제거: 명시적 `#ff00ff` 키·소프트 매트·디스필·1px 가장자리 수축
- 모션: `256x256` 셀, 캐릭터 높이 `180px`, 기준선 `y=244`
- UI: 버스트 `512px`, 패널 초상화 `128px`, 작은 위젯 아바타 `48px`
