# 흔함 등급 생성 원본

`docs/characters/1-common.md`의 외형·공격 묘사를 기준으로 imagegen 내장 도구에서 생성한 원본이다.

## 공통 프롬프트 규격

- 용도: `stylized-concept`, 프로덕션용 32비트 시대 픽셀아트 게임 에셋
- 스타일: 생활감 있는 낮은 채도, 풍부한 단계 명암, 고정된 어두운 외곽선, 동일 픽셀 밀도
- 비율: 인간형 1:3.5 두신, 조종사+드론도 같은 70% 높이 바운딩 박스
- 색상: 흔함 회색 `#9aa0a6` 중심의 실용적인 도시 생활 팔레트
- 배경: 균일한 `#ff00ff` 마젠타 크로마키, 피사체에는 마젠타 사용 금지
- 금지: 글자·숫자·로고·워터마크·격자선·바닥·그림자·반사
- 마스터: 좌측 고품질 버스트, 우측 정면·후면·측면 전신
- 공격: 2행×4열, 준비·장전·발동·후속·복귀가 드러나는 8프레임
- 이동: 정면·후면·측면 각 8프레임, 동일 크기와 기준선

## 이번 교체 대상

| ID | 외형 핵심 | 공격 8프레임 | 이동 핵심 |
| --- | --- | --- | --- |
| `dv1` | 헬멧·배달통·음식 봉투·지친 눈 | 포장 음식 수평 투척 | 배달통이 흔들리는 빠른 보행 |
| `pc1` | 앳된 얼굴·큰 제복·삐뚤어진 모자·테이저 | 긴장한 테이저 조준과 발사 | 경직된 순찰 보행 |
| `sn1` | 후드티·스포츠 아대·Y자 새총·구슬 주머니 | 구슬 장전·당김·발사 | 새총을 느슨하게 든 보행 |
| `dr1` | 안경·아웃도어 조끼·조종기·쿼드콥터 | 드론 조준·펠릿 사출 | 조종사와 드론의 동기 이동 |
| `dm1` | 안전모·안전조끼·체크 남방·대형 망치 | 망치 수직 내려찍기 | 망치를 어깨에 멘 중량 보행 |
| `tf1` | 챙 모자·관리 조끼·집게·빗자루 | 집게 찌르기와 빗자루 스윙 | 두 도구를 든 순찰 보행 |

캐릭터 일관성 검수에서 `dv1`, `tf1` 이동과 `pc1` 공격은 마스터 시트를 참조 이미지로 넣어 재생성했다.

## 런타임 변환

- 크로마 제거: imagegen 스킬의 `remove_chroma_key.py`, 명시적 `#ff00ff` 키·소프트 매트·디스필·1px 가장자리 수축
- 모션: `256x256` 셀, 캐릭터 높이 `180px`, 기준선 `y=244`
- UI: 버스트 `512px`, 패널 초상화 `128px`, 작은 위젯 아바타 `48px`
- 최종 파일: `public/sprites/common/`

`fl1`, `fb1`, `hp1`, `hc1`은 다음 흔함 교체 단계에서 같은 규격으로 갱신한다.
