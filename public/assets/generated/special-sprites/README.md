# 특별함 등급 생성 원본

`docs/characters/3-special.md`의 외형·공격 묘사를 기준으로 imagegen 내장 도구에서 생성한 원본이다.

## 공통 생성 규격

- 전문 장비가 읽히는 32비트 픽셀아트, 파랑 `#42a5f5` 포인트와 절제된 이펙트
- 인간형 1:3.5 두신, 헬기도 같은 70% 높이 바운딩 박스
- 마스터: 고품질 버스트 + 정면·후면·측면 전신
- 기본 공격: 2행×4열 8프레임, 이동: 정면·후면·측면 각 8프레임
- 균일한 `#ff00ff` 크로마키 배경, 글자·격자선·바닥·그림자 없음

## 유닛별 핵심

| ID | 외형 | 기본 공격 | 이동 |
| --- | --- | --- | --- |
| `dv3` | 셔츠·파란 넥타이·걸친 재킷·헬멧·스마트폰 | 콜 배정과 노룩 봉투 투척 | 폰을 확인하는 사장 보행 |
| `pc3` | 검은 SWAT 장비·바이저·고무탄 발사기 | 전진 고무탄 2점사 | 낮은 전술 이동 |
| `sn3` | 회색 전술복·대형 저격총·허리 새총 | 스탠딩 조준·격발·볼트 조작 | 장총을 통제한 순찰 보행 |
| `dr3` | 헤드셋·파란 스카프·소형 정찰 헬기 | 저공 호버 광역 사격 | 3방향 헬기 호버 이동 |
| `dm3` | 카키 방폭복·파란 패널·니퍼·폭탄 캐리어 | 폭약 설치·후퇴·컴팩트 기폭 | 방폭복 중량 보행 |

`dr3` 전투 프레임은 파일럿이 캐노피 안에 탑승한 헬기 일체형으로 정리했다. `dm3` 이동 원본은 7개 고유 자세에 첫 자세를 복귀 프레임으로 연결했다. 기존 스킬·패시브 이펙트 시트는 별도 경로를 유지한다.
