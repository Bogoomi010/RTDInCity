# 전설 등급 생성 원본

`docs/characters/5-legendary.md`의 외형·공격 묘사를 기준으로 imagegen 내장 도구에서 생성한 알파 원본이다.

## 프롬프트 규격

- 마스터: 버스트 + 정면/후면/측면 전신, 전설 금색 `#ffa726` 포인트
- 공격: 준비·예비동작·발동·후속동작·복귀가 드러나는 8프레임
- 이동: 정면/후면/측면 각 8프레임
- 배경: 균일한 `#00ff00` 크로마키, 피사체에는 초록색 사용 금지
- 스타일: 32비트 시대 픽셀아트, 1:3.5 두신, 동일 픽셀 밀도·외곽선

런타임 출력은 `256x256` 셀, 캐릭터 높이 `180px`, 기준선 `y=244`로 통일했다.
UI 출력은 버스트 `512px`, 패널 `128px`, 작은 위젯 `48px` 세 종류다.

## 특수 원본 처리

- 모든 공격 원본은 긴 소품과 이펙트를 위해 2x4 격자로 만들고 `--attack-rows 2`를 사용한다.
- `dr5` 이동 원본은 실제 4행x7포즈를 검출해 정면/후면/측면 행을 선택하고 첫 포즈를 복귀 프레임으로 추가한다.
- `dm5` 이동 원본도 방향별 7포즈를 같은 방식으로 8프레임 루프로 만든다.
- 모델이 그린 얇은 셀 구분선은 연결 영역의 크기와 밀도로 판별해 제외한다.

최종 런타임 파일은 `public/sprites/legendary/`에 있다.
